# FZAstro AI Astro Web Assets

This folder contains web/static assets used by the migrated FZASTRO astronomy tools embedded in FZAstro AI.

These files are bundled with the Windows EXE so the ASTRO tools can render supporting pages, images, and local resources without depending on the old standalone project layout.

Do not place unrelated application documentation in this folder. The root `README.md` is the canonical project documentation.
